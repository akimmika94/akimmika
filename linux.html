Groupes
Gestion des groupes
addgroup, usermod, gpasswd, id, getent
▼ 
# Créer un groupe
sudo addgroup jedis

# Ajouter user dans un groupe (sans retirer des autres)
sudo usermod -aG jedis vador

# Ajouter user ET retirer de tous les autres groupes
sudo usermod -g jedis vador

# Retirer user d'un groupe
sudo gpasswd --delete vador jedis

# Voir les groupes d'un utilisateur
id vador

# Voir les membres d'un groupe
getent group jedis

# Supprimer un groupe
sudo delgroup jedis
-aG (Append Group) conserve les groupes existants. Sans -a, l'utilisateur est retiré de tous ses autres groupes !

chmod +x script.sh      # exécutable pour tous

# Lire les droits
ls -l
# -rwxrw-r-- 1 vador empire 33 ... fichier
#  ^^^        = droits u(rwx) g(rw) o(r)
Exemple -rwxrw-r-- :
- fichier | rwx propriétaire | rw- groupe | r-- autres

  Ida      INT AUTO_INCREMENT,

  FOREIGN KEY (Idj) REFERENCES journaliste(Idj)

▼
Carte réseau : enp0s3=NAT(Internet), enp0s8=192.168.240.254, enp0s9=192.168.57.254
# Étape 1 – Activer le routage IP
sudo nano /etc/sysctl.conf -l
# Décommenter la ligne 28 :
net.ipv4.ip_forward=1
sudo sysctl -p    # vérifier

# Étape 2 – Créer la table NAT (nft)
sudo nft add table NAT
sudo nft add chain NAT POST-RT '{ type nat hook postrouting priority 0 ;}'
sudo nft add rule NAT POST-RT oif enp0s3 masquerade

# Étape 3 – Sauvegarder les règles (en root)
su -
nft list ruleset > /etc/nftables.conf
systemctl enable nftables.service

# Commandes de diagnostic NFT
nft list ruleset    # afficher les règles
nft flush ruleset   # supprimer toutes les règles
nft -c -f /etc/nftables.conf   # vérifier la syntaxe

Script users
Script import d'utilisateurs depuis CSV

▼
Fichier users.txt : alex:sudo:Reponse2025 (1 ligne par user)
#!/bin/bash
fichier=/home/tssr/scripts/users.txt

# Étape 1 : vérifier que le fichier existe
if ! [[ -f ${fichier} ]]; then
  echo "Fichier users.txt introuvable, abandon."
  exit
fi

# Étape 2 : confirmer l'import
echo "Importer les utilisateurs ? (o/n)"
read reponse
if [[ ${reponse} != "o" ]]; then exit; fi

# Étape 3 : créer les utilisateurs
for vligne in $(cat ${fichier}); do
  login=$(echo ${vligne}     | cut -d':' -f 1)
  groupe=$(echo ${vligne}    | cut -d':' -f 2)
  motdepasse=$(echo ${vligne}| cut -d':' -f 3)

  sudo useradd ${login} --home-dir /home/${login}/ --create-home
  echo "${login}:${motdepasse}" | sudo chpasswd

  if [[ ${groupe} = 'sudo' ]]; then
    sudo usermod -aG sudo ${login}
  fi
done

# Lancer en root :
# sudo bash import_users.sh
