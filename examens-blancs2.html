b
Corriger le script PowerShell C:\scripts\users.ps1 pour créer 5 stagiaires dans l'AD
Script corrigé
$DomainName = "examen.labo" $ListUsers = "Stagiaire1","Stagiaire2","Stagiaire3","Stagiaire4","Stagiaire5" $SecurePassword = ConvertTo-SecureString "Reponse2025" -AsPlainText -Force foreach ($UserName in $ListUsers) { New-ADUser -Name $UserName -SamAccountName $UserName ` -UserPrincipalName "$UserName@$DomainName" ` -AccountPassword $SecurePassword -Enabled $true ` -Path "OU=informatique,OU=examen,DC=examen,DC=labo" } # Vérification Get-ADUser -Filter * -SearchBase "OU=informatique,OU=examen,DC=examen,DC=labo"
⚠ Points de correction fréquents : chemin OU incorrect, guillemets manquants autour du domaine, -Enabled $true absent.
